debug lacp 
-----------

**Command syntax: debug lacp**

**Description:** Debug LACP events

**CLI example:**
::

	dnRouter# configure
	dnRouter(cfg)# debug lacp
	
	dnRouter(cfg)# no debug lacp
	
	dnRouter# set debug lacp
	
	dnRouter# no set debug lacp
	
	
	
**Command mode:** operational/config

**TACACS role:**

TACACS role "operator" - debug logging is a persistent configuration
