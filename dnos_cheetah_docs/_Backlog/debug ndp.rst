debug ndp 
----------

**Command syntax: debug ndp**

**Description:** Debug NDP events

**CLI example:**
::

	dnRouter# configure
	dnRouter(cfg)# debug ndp
	
	dnRouter(cfg)# no debug ndp
	
	dnRouter# set debug ndp
	
	dnRouter# unset debug ndp
	
	
	
**Command mode:** operational/config

**TACACS role:**

TACACS role "operator" - debug logging is a persistent configuration
