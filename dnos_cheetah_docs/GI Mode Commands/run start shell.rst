run start shell
---------------

**Minimum user role:** viewer

Provides access to the white box shell.

**Command syntax: run start shell**

**Command mode:** GI

.. **Note**

 - The "exit" command reverts to GI Mode.

**Example**
::

	dnRouter(GI)# run start shell
	root@WDV1947700016:/#

	root@WDV1947700016:/# exit
	dnRouter(GI)#
