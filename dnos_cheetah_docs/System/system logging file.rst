system logging file
-------------------

**Minimum user role:** operator

To select a file type for configuring system events log:

**Command syntax: file [file-type] [parameters]**

**Command mode:** config

**Hierarchies**

- system logging


**Note**

- Notice the change in prompt.

- After selecting the file type, you can proceed to configure the parameters for that file type.

.. - no command reverts the log files parameter to its default value

**Parameter table**

+-----------+-----------------------------------------------------------------------------------------------------------------------------------------------------+---------------+---------+
| Parameter | Description                                                                                                                                         | Range         | Default |
+===========+=====================================================================================================================================================+===============+=========+
| file-type | The type of the log file                                                                                                                            | system-events | \-      |
+-----------+-----------------------------------------------------------------------------------------------------------------------------------------------------+---------------+---------+
| Size      | The maximum size of the file. When the size is reached, new logs will be saved in a new file.                                                       | 1..1024 MB    | 10 MB   |
+-----------+-----------------------------------------------------------------------------------------------------------------------------------------------------+---------------+---------+
| Files     | The maximum number of log files that the system will create. When the number is reached and the last file is full, the oldest file will be deleted. | 1..20         | 10      |
+-----------+-----------------------------------------------------------------------------------------------------------------------------------------------------+---------------+---------+

**Example**
::

	dnRouter# configure
	dnRouter(cfg)# system
	dnRouter(cfg-system)# logging
	dnRouter(cfg-system-logging)# file system-events
	dnRouter(cfg-system-logging-file)# files 20
	dnRouter(cfg-system-logging-file)# size 3 
	

	
	
	

**Removing Configuration**

To revert the router-id to default: 
::

	dnRouter(cfg-system-logging-file)# no files 
	dnRouter(cfg-system-logging-file)# no size
	dnRouter(cfg-system-logging)# no file

.. **Help line:** configure parameters for log files generated by DNOS.
