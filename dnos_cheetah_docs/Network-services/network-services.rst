network-services
----------------

**Minimum user role:** operator

Enter the network-services configuration hierarchy.

**Command syntax: network-services**

**Command mode:** config

**Example**
::

    dnRouter# configure
    dnRouter(cfg)# network-services
    dnRouter(cfg-netsrv)#


**Removing Configuration**

To remove services for non-default VRFs:
::

    no network-services
