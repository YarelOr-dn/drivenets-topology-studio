multicast
---------

**Minimum user role:** operator

To configure the system to start using multicast routing.

**Command syntax: multicast**

**Command mode:** config

**Hierarchies**

- protocols multicast

**Example**
::

	dnRouter# configure
	dnRouter(cfg)# multicast
	dnRouter(cfg-multicast)#

.. **Help line:** Enters the Multicast configuration hierarchy.
