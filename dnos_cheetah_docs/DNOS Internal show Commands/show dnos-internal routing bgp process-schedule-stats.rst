show dnos-internal routing bgp process-schedule-stats
-----------------------------------------------------

**Minimum user role:** viewer

To display BGP processing blocks statistics:

**Command syntax: show dnos-internal routing bgp process-schedule-stats**

**Command mode:** operation

**Example**
::

	dnRouter# show dnos-internal routing bgp process-schedule-stats
	Total yields: 335
	Total runs: 542
	Total steps: 5817998
	AVG #steps per run: 10734
	Best granularity: 185472
	Current granularity: 10308
