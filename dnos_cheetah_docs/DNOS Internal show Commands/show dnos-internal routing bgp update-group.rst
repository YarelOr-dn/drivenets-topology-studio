show dnos-internal routing bgp update-group
-------------------------------------------

**Minimum user role:** viewer

To display BGP update groups:

**Command syntax: show dnos-internal routing bgp update-group**

**Command mode:** operation

**Example**
::

	dnRouter# show dnos-internal routing bgp update-group
	VRF default IPv4 Unicast:

	ID:1 status:active lock:23000 using peers count:500
	Sort BGP_PEER_EBGP
	AS-loop-check
	AS4 Capable
