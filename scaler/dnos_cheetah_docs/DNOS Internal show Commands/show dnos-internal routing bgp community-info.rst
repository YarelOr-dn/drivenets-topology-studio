show dnos-internal routing bgp community-info
-----------------------------------------------

**Minimum user role:** viewer

To display all BGP community information:

**Command syntax:show dnos-internal routing bgp community-info** 

**Command mode:** operation

**Example**
::

	dnRouter# show dnos-internal routing bgp community-info
	Address          Refcnt   Community
	[0x55e440380310] (100)    internet

.. **Help line:** List all bgp community information
