system logging 
---------------

**Minimum user role:** operator

To configure parameters for the "system-events.log" file generation by DNOS:

**Command syntax: logging**

**Command mode:** config

**Hierarchies**

- system

.. **Note**

 - no command reverts the system events logging parameter to its default value

**Parameter table**

+-----------+------------------------------------------+-----------------+---------+
| Parameter | Description                              | Range           | Default |
+-----------+------------------------------------------+-----------------+---------+
| file-type | The type of file to be generated by DNOS | "system events" | \-      |
+-----------+------------------------------------------+-----------------+---------+
| files     | number of files for specific log file    | 1..20           | 10      |
+-----------+------------------------------------------+-----------------+---------+
| size      | maximum file size for specific log file  | 1..1024MB       | 10MB    |
+-----------+------------------------------------------+-----------------+---------+

**Example**
::

	dnRouter# configure
	dnRouter(cfg)# system
	dnRouter(cfg-system)# logging

**Removing Configuration**

To revert to the default value:
::

	dnRouter(cfg-system)# no logging



.. **Help line:** configure parameters for "system-events.log" file generation by DNOS.
