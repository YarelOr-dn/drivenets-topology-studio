system network-resources
------------------------

**Minimum user role:** operator

To network-resources hierarchy:

**Command syntax: network-resources**

**Command mode:** config

**Hierarchies**

- system

**Example**
::

    dnRouter# configure
    dnRouter(cfg)# system
    dnRouter(cfg-system)# network-resources
    dnRouter(cfg-system-netres)#


**Removing Configuration**

To delete all network-resources:
::

    dnRouter(cfg-system)# no network-resources
