show system commit
------------------

**Minimum user role:** viewer

To view the current commit confirm activity on the system, use the following command:



**Command syntax: show system commit**

**Command mode:** operational



**Example**
::

	dnRouter# show system commit
	[27-Jan-2017 22:11:00 UTC] admin via cli commit confirm, rollback in 9 minutes
	
	dnRouter# show system commit
	no commits

.. **Help line:** display current commit confirm activity or running commit by other users
