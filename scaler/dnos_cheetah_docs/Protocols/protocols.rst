protocols
---------

**Minimum user role:** operator

Enter protocols configuration mode

**Command syntax: protocols**

**Command mode:** config

**Note:**

'no protocols' - remove all protocols configuration

**Example**
::

	dnRouter# configure
	dnRouter(cfg)# protocols
	dnRouter(cfg-protocols)#

	dnRouter(cfg)# no protocols
