forwarding-options
------------------

**Minimum user role:** operator

To enter the forwarding-options configuration hierarchy:

**Command syntax: forwarding-options**

**Command mode:** config

**Example**
::

    dnRouter# configure
    dnRouter(cfg)# forwarding-options
    dnRouter(cfg-fwd_opts)#


**Removing Configuration**

To remove the forwarding-options configuration:
::

    dnRouter(cfg)# no forwarding-options
