tracking-policy
---------------

**Minimum user role:** operator

To enter the tracking-policy configuration hierarchy:

**Command syntax: tracking-policy**

**Command mode:** config

**Example**
::

    dnRouter# configure
    dnRouter(cfg)# tracking-policy
    dnRouter(cfg-trkpl)#


**Removing Configuration**

To remove the forwarding-options configuration:
::

    dnRouter(cfg)# no tracking-policy
