services
--------

**Minimum user role:** operator

Enter the services configuration hierarchy.

**Command syntax: services**

**Command mode:** config

**Example**
::

    dnRouter# configure
    dnRouter(cfg)# services
    dnRouter(cfg-srv)#


**Removing Configuration**

To remove services:
::

    dnRouter(cfg)# no services
