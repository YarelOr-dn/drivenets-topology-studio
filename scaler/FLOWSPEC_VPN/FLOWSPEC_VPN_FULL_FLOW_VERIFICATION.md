## Spirent Configuration: FlowSpec Rules + Traffic

### FlowSpec Rules Advertised by ARBOR_SIM (VLAN 10)

| Rule Name | Match Criteria | Action | Route Target |
|-----------|----------------|--------|--------------|
| **DROP-HTTP** | dst: 10.10.10.1/32, protocol: TCP (6), dport: 80 | traffic-rate: 0 (DROP) | 65000:100 |
| **DROP-ICMP** | dst: 10.10.10.1/32, protocol: ICMP (1) | traffic-rate: 0 (DROP) | 65000:100 |
| **RATE-DNS** | dst: 10.10.10.2/32, protocol: UDP (17), dport: 53 | traffic-rate: 1000000 (1Mbps) | 65000:100 |

#### FlowSpec NLRI Format (as seen in BGP)

```
Rule: DROP-HTTP
  NLRI: DstPrefix:=10.10.10.1/32,Protocol:=6,DstPort:=80
  Extended Community: RT:65000:100, traffic-rate:0

Rule: DROP-ICMP  
  NLRI: DstPrefix:=10.10.10.1/32,Protocol:=1
  Extended Community: RT:65000:100, traffic-rate:0

Rule: RATE-DNS
  NLRI: DstPrefix:=10.10.10.2/32,Protocol:=17,DstPort:=53
  Extended Community: RT:65000:100, traffic-rate:1000000
```
